
# Frontend Challenge Chicks Gold

Mock design




## Run Locally

### Clone the project

```bash
  git clone https://github.com/ClaudioMFCDev/Frontend-Challenge
```

### Go to the project directory

```bash
  cd frontend\frontend-challenged
```

### Install dependencies

```bash
  npm install
```

### Start the server

```bash
  npm start
```
